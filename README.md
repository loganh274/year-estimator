# Manual methods
How to Run Training:

    Install Dependencies: pip install -r requirements.txt

    Start Training: python src/train.py

    Enter Credentials: Follow the on-screen prompts to enter your AWS details.

How to run trained model:
    launch app.py using terminal: 
        cd '\PARENT_FOLDER\year-estimator'
        python app.py
